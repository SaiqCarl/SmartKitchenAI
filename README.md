🍳 Smart Kitchen AI

Smart Kitchen AI is an AI-powered cooking assistant that helps users discover delicious dishes using the ingredients already available in their kitchen. The system combines modern UI/UX design, local AI processing, and online recipe integration to generate smart recipe recommendations, cooking instructions, and cuisine-based meal suggestions in real time.

Built using Python, Streamlit, LM Studio, and Llama 3, the application demonstrates how artificial intelligence can be integrated into practical real-world kitchen and meal-planning scenarios.

✨ Features
🥬 Ingredient-based recipe recommendations
🤖 AI-generated cooking instructions
🍽️ Cuisine preference selection
🔥 Cooking method filtering
🇵🇭 Filipino cuisine prioritization
🌐 Online recipe search integration
⚡ Real-time AI responses
🎨 Modern and responsive UI/UX
🧠 Local AI processing using LM Studio + Llama 3
💡 Cooking tips and ingredient alternatives

🧠 AI Integration

The application uses:

LM Studio for local AI hosting
Llama 3 as the AI model
TheMealDB API for online recipe searching

This allows the system to generate intelligent cooking recommendations without relying entirely on cloud-based AI services.
